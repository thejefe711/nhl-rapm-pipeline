# Validation Runbook

## Overview
This document describes how to run the Data Quality and Validation Framework for the NHL RAPM pipeline.

## Running Validations

### Full Validation Suite
To run all validations for a specific season:
```bash
python -m nhl_pipeline.src.validation.run_all_validations --season 2024 --db-path nhl_canonical.duckdb
```

### Running Tests
Unit tests:
```bash
pytest nhl_pipeline/tests/unit
```

Integration tests:
```bash
pytest nhl_pipeline/tests/integration
```

Golden file tests:
```bash
pytest nhl_pipeline/tests/golden
```

## Monitoring
Check the `validation_report.json` generated by the runner for detailed results.
Use `src/monitoring/dashboard_queries.sql` to query the database for health trends.
